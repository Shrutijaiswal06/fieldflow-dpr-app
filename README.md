# Construction Field Management App

## Tech Stack

* React.js (Vite)
* Tailwind CSS
* React Router v6
* JavaScript

## Features

* Login authentication (mock)
* Project dashboard with project cards
* Daily Progress Report (DPR) form
* Image upload with preview (max 3 images)
* Responsive design for mobile, tablet, and desktop
* Input validation with user-friendly messages
